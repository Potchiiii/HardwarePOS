PROCUREMENT MODULE - QUICK START GUIDE
========================================

STEP 1: Initialize the Database
--------------------------------
Open your browser and navigate to:
http://localhost/Hardware/procurement/setup.php

This will create the necessary database tables:
- purchase_orders
- notifications

STEP 2: Access the Procurement Module
--------------------------------------
1. Login with a procurement user account
   - Username: Halon (or any user with user_type = 'procurement')
   
2. You'll be redirected to the procurement dashboard

STEP 3: Basic Workflow
----------------------

A. MONITOR LOW STOCK:
   1. Go to "Low Stock Items" in the sidebar
   2. See all items below their thresholds
   3. Click "Order" button to create a purchase order

B. CREATE PURCHASE ORDERS:
   1. From "Low Stock Items", click "Order"
   2. Enter quantity to order
   3. Add notes if needed
   4. Click "Create Order"

C. MANAGE ORDERS:
   1. Go to "Purchase Orders"
   2. View all orders with their statuses
   3. Click "Status" button to update order status:
      - Pending → Approved → Ordered → Received
   4. Use filters to view orders by status

D. NOTIFY INVENTORY STAFF:
   1. Go to "Notify Staff"
   2. Click "+ New Notification"
   3. Enter:
      - Order number (e.g., ORD-2025-001)
      - Item name
      - Quantity received
      - Message for staff
   4. Click "Create"
   5. Click "Send" to notify inventory staff
   6. Staff will see notification in their inventory dashboard

FEATURES SUMMARY
================

✓ Dashboard
  - Quick overview of low stock, pending orders, and inventory

✓ Low Stock Items
  - View items that need ordering
  - Search items
  - Create orders with one click
  - Color-coded stock indicators

✓ Purchase Orders
  - Track all orders
  - Update status (Pending → Approved → Ordered → Received)
  - Filter by status
  - Delete orders
  - Statistics by status

✓ Notify Staff
  - Create notifications for inventory staff
  - Mark notifications as sent
  - Filter by sent/pending status
  - Delete notifications
  - Track notification history

KEYBOARD SHORTCUTS
==================
- Search in tables using the search boxes
- Filter orders using the status buttons
- Use modals to quickly create/update records

TIPS
====
1. Update order status as soon as it changes
2. Send notifications immediately when stock arrives
3. Use meaningful messages in notifications so staff understands what arrived
4. Delete old notifications to keep the list clean

For any issues, check the browser console (F12) for error messages.
